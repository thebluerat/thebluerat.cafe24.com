Skill
HTML

웹 표준과 웹 접근성을 준수하는 HTML 정보 구조 설계로 퍼블리싱을 합니다. 구조와 구성을 중시하는 HTML5의 마크업 방식과 시맨틱 태그를 사용해 레이아웃을 설계합니다. 퍼블리싱하기 전, 클래스 네임과 HTML 정보 구조를 설계한 후 퍼블리싱 작업을 진행합니다. 폰트 서식, 컬러, 사이즈, 반응형, 상호 작용, HTML 와이어프레임을 미리 그리고 메모한 후 체계적으로 수행합니다.

CSS

CSS는 태그의 용도와 의미를 정확하게 파악하고 사용하도록 하고 있습니다. inline 요소인 a 태그에 적용이 안 되는 상하 margin을 굳이 부여하지 않고 span 태그는 텍스트에만, button 태그는 버튼에만 사용하는 등 정확하고 효율적으로 사용합니다.

Javascript

내비게이션, 탭 메뉴, 어코디언 등 웹 퍼블리싱에 필수적이라고 할 수 있는 상호 작용을 위한 작업은 플러그인을 사용하지 않고 제이쿼리 하드코딩으로 제작합니다. 필수적인 자바스크립트를 활용할 수 있습니다.

Responsive Web

float, position, media query를 활용한 반응형 웹 제작을 할 수 있습니다. flex와 grid 방식으로도 반응형 웹 사이트를 제작할 수 있습니다.

Mobile Web App

모바일 환경을 우선으로 생각하는 모바일 웹 앱을 만들 수 있습니다. 사용자 경험에 도움을 주기 위한 효율적인 컨텐츠 배치를 우선으로 생각합니다. 모바일 대응을 위한 플랫폼으로 반응형 웹, 모바일 웹, 모바일 웹 앱, 네이티브 앱에 대해 이해하고 있습니다. 사용자에게 최적화하는 방법을 배우기 위해 디자인 아티클을 구독하여, '터치 스크린에 알맞은 탭 타겟의 최소 크기, 터치 프렌들리 디자인, 특정 상호작용을 위해 사용자가 스마트폰을 잡는 방식' 등 일상적으로 관심을 가지고 배우고 있습니다.

Text Editor

텍스트 에디터는 Visual Studio Code를 주로 사용합니다. Brackets와 Atom으로 퍼블리싱 공부를 시작하여 익숙하고, Java Spring을 공부할 때는 IDE Eclipse를 주로 사용했습니다.Sublime Text와 EditPlus 사용 경험이 있습니다.
